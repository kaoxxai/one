02. 數學模型與指標公式

2.1 熱度水準與加速度 (Heat Dynamics)

利用新聞篇數與搜尋量計算標準化熱度 (Z-Score) 及其變動率。

$$ HeatLevel_{i,t} = \frac{N_{i,t} - MA_{N}(N_{i,t}, L_N)}{Std_{N}(N_{i,t}, L_N)} $$
熱度斜率 (Slope)
$$ HeatSlope_{i,t} = N_{i,t} - N_{i,t-k} $$
熱度加速度 (Accel)
$$ HeatAccel_{i,t} = HeatSlope_{i,t} - HeatSlope_{i,t-k} $$

2.2 價格與波動 (Price & Volatility)

# 創新高指標 (PriceNewHigh)

$$ PriceNewHigh_{i,t} = \begin{cases} 1, & P_{i,t} = \max(P_{i,t-N+1}, \dots, P_{i,t}) \\ 0, & \text{otherwise} \end{cases} $$

# 波動率爆發 (VolSpike)

短期波動率相對於長期波動率的倍數。

$$ Volatility_{i,t} = Std(r_{i,t}, \dots, r_{i,t-L_{vol}}) $$ $$ VolSpike_{i,t} = \frac{Volatility_{i,t}}{MA(Volatility_{i,t}, L_{base})} $$

2.3 籌碼流向 (Flow Analytics)

主力淨買超比率
$$ Flow_{i,t} = \frac{F_{i,t}}{V_{i,t}} $$

定義 FlowTurn 為「短期均線 > 0 且 長期均線 < 0」（或由正轉負的轉折點，視策略方向而定）。

$$ FlowTurn_{i,t} = \mathbb{1}\big( MA(Flow_{i,t}, L_{in}) > 0 \land MA(Flow_{i,t}, L_{out}) < 0 \big) $$

2.4 Theme Maturity (綜合分數)

TM 指標用來判斷主題是否過熱或即將反轉。

Component Logic Meaning
Plateau $\mathbb{1}(Heat > \theta_1 \land |Slope| < \theta_2)$ 熱度高檔鈍化
PriceHot $\mathbb{1}(NewHigh \land VolSpike > \theta_3)$ 價格創高且波動放大
FlowWeak $\mathbb{1}(FlowTurn == 1)$ 籌碼動能衰退
$$ TM_{i,t} = Plateau_{i,t} + PriceHot_{i,t} + FlowWeak_{i,t} $$
TM = 3 High Risk / Mature
TM = 0 Early Stage / Safe

03. Python 因子計算 (Pandas Implementation)

pandas >= 1.3.0 
import pandas as pd
import numpy as np

# 假設 df 為 MultiIndex DataFrame: index=date, columns=(stock_id, field)
# fields: 'close', 'volume', 'news_count', 'search_index', 'flow'

def rolling_zscore(x, window):
    """計算滾動 Z-Score"""
    ma = x.rolling(window).mean()
    std = x.rolling(window).std()
    # 避免除以零
    return (x - ma) / std.replace(0, np.nan)

def compute_factors(df, L_N=60, k=5, L_vol=20, L_base=60):
    """
    計算 Theme Maturity 相關因子
    """
    # 提取各欄位數據 (Cross-section view)
    close = df.xs('close', axis=1, level=1)
    volume = df.xs('volume', axis=1, level=1)
    news = df.xs('news_count', axis=1, level=1)
    search = df.xs('search_index', axis=1, level=1)
    flow = df.xs('flow', axis=1, level=1)

    # --- 1. 熱度因子 ---
    news_z = rolling_zscore(news, L_N)
    search_z = rolling_zscore(search, L_N)
    # 綜合熱度 (等權重)
    heat_level = 0.5 * news_z + 0.5 * search_z
    
    # 熱度動能
    heat_slope = news.diff(k)
    heat_slope_abs = heat_slope.abs()
    # heat_accel = heat_slope.diff(k) # Optional

    # --- 2. 價格與波動 ---
    # Log Returns
    ret = np.log(close / close.shift(1))
    
    # Volatility Spike
    vol = ret.rolling(L_vol).std()
    vol_base = vol.rolling(L_base).mean()
    vol_spike = vol / vol_base

    # 創新高 (New High) - 60日新高
    price_new_high = (close == close.rolling(60).max()).astype(int)

    # --- 3. 籌碼因子 ---
    # 淨流量比率
    flow_ratio = flow / volume.replace(0, np.nan)
    
    flow_short = flow_ratio.rolling(5).mean()
    flow_long = flow_ratio.rolling(20).mean()
    
    # 籌碼轉弱 (短期 > 0 且 長期 < 0，或根據原邏輯調整轉折定義)
    # 原始邏輯：FlowTurn (由多轉空 or 特定型態)
    # 這裡實作：短期均線轉負的瞬間 (範例)
    flow_turn = ((flow_short > 0) & (flow_long < 0)).astype(int)

    # --- 4. 綜合訊號 (Theme Maturity) ---
    # 設定閥值 (Thresholds)
    TH_HEAT = 1.0
    TH_SLOPE = 0.01
    TH_VOL = 1.5

    # Plateau: 熱度高但斜率趨緩
    plateau = ((heat_level > TH_HEAT) & (heat_slope_abs < TH_SLOPE)).astype(int)
    
    # PriceHot: 創新高且波動放大
    price_hot = ((price_new_high == 1) & (vol_spike > TH_VOL)).astype(int)
    
    # FlowWeak: 籌碼轉弱訊號
    flow_weak = flow_turn

    # TM Score (0-3)
    tm_score = plateau + price_hot + flow_weak

    # --- 整理輸出 ---
    factors = {
        'heat_level': heat_level,
        'vol_spike': vol_spike,
        'tm_score': tm_score
    }
    
    # 將 Dictionary 轉換回 MultiIndex DataFrame
    output_frames = []
    for name, data in factors.items():
        # 為 columns 增加一層 level
        data.columns = pd.MultiIndex.from_product([data.columns, [name]])
        output_frames.append(data)
    
    return pd.concat(output_frames, axis=1).sort_index(axis=1)

04. 排序回測框架 (Ranking & Backtest) 

def backtest_portfolio(factor_df, close_df, rebal_freq='M', n_q=5):
    """
    簡易多空投組回測
    Long: TM Score 最低 (Theme Early)
    Short: TM Score 最高 (Theme Mature)
    """
    # 取得 TM 因子與 Resample 日期
    tm = factor_df.xs('tm_score', axis=1, level=1)
    rebal_dates = tm.resample(rebal_freq).last().index
    
    # 使用明日報酬 (因為是收盤後計算訊號)
    daily_ret = close_df.pct_change().shift(-1)
    
    portfolio_returns = []

    for date in rebal_dates:
        if date not in tm.index: continue
            
        # 取得當日因子值並去除空值
        scores = tm.loc[date].dropna()
        if scores.empty: continue
            
        # 分組 (Quantiles)
        try:
            labels = pd.qcut(scores, n_q, labels=False, duplicates='drop')
        except ValueError:
            continue # 樣本不足
            
        # 定義多空名單
        long_stocks = scores.index[labels == 0]       # Q1: Low TM
        short_stocks = scores.index[labels == n_q-1]  # Q5: High TM
        
        # 計算持有期間 (直到下一次 Rebalance)
        # 此處簡化為持有至月底
        next_idx = tm.index.get_loc(date) + 1
        if next_idx >= len(tm.index): break
            
        # 計算該月平均報酬 (Equal Weight)
        # 實際應計算每日報酬序列
        long_ret = daily_ret.loc[date, long_stocks].mean()
        short_ret = daily_ret.loc[date, short_stocks].mean()
        
        # Long-Short Spread
        portfolio_returns.append({
            'date': date,
            'long_ret': long_ret,
            'short_ret': short_ret,
            'spread': long_ret - short_ret
        })

    return pd.DataFrame(portfolio_returns).set_index('date')

Strategy Logic

Long (Q1)
TM Score = 0. 主題初期，尚未過熱。
Short (Q5)
TM Score = 3. 主題成熟，伴隨籌碼渙散與波動放大。
