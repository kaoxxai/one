# **🍃 ESG 詞彙學習網頁使用說明書**

**ESG Vocabulary Learning Webpage User Guide**

## **1\. 簡介 (Introduction)**

本系統專為 ESG (環境、社會、治理)、Quant (量化分析) 及 Data (數據) 領域的學習者設計。透過互動式卡片、分類檢索與隨機測驗，協助您快速掌握專業術語的中英文對照與核心定義。

## **2\. 快速導覽 (Interface Overview)**

進入網頁後，主要介面分為三個區域：

1. **頂部導航列 (Top Navigation)**  
   * **左側 Logo**：點擊可隨時「回到主頁 (Home)」。  
   * **測驗挑戰 (Quiz)**：點擊進入測驗選單。  
   * **切換學習模式 (Toggle Mode)**：在「列表模式」與「單張卡片學習模式」間切換。  
2. **控制工具列 (Control Bar)**  
   * **搜尋框**：輸入中文或英文關鍵字（如 "Carbon", "碳"）。  
   * **分類按鈕**：可滑動選擇不同類別（Environmental, Social, Governance, ESG Quant... 等）。  
3. **主內容區 (Main Content)**  
   * 顯示學習卡片或測驗內容。

## **3\. 功能操作詳解**

### **A. 列表瀏覽模式 (Grid View)**

這是系統預設的首頁模式，適合快速瀏覽大量詞彙。

* **瀏覽卡片**：所有詞彙以卡片陣列呈現。  
* **翻轉卡片**：點擊任一張卡片，卡片會翻轉至背面。  
  * **正面**：顯示中文詞彙、類別標籤。  
  * **背面**：顯示英文術語、詳細解釋。  
* **進入學習**：卡片背面右下角有「**進入學習模式**」按鈕，點擊後會直接跳轉至該單字的沉浸式學習畫面。

### **B. 沉浸式學習模式 (Flashcard Mode)**

此模式一次只顯示一張大卡片，適合專注記憶與練習發音。

* **切換方式**：點擊頂部的「切換學習模式」按鈕，或從列表模式的單張卡片進入。  
* **翻轉查看**：點擊大卡片中央（或按鍵盤 空白鍵），查看中英對照與定義。  
* **發音功能**：卡片背面右上角有 **喇叭圖示 (🔊)**，點擊可聆聽英文術語的標準發音。  
* **切換卡片**：  
  * 點擊螢幕下方的「上一個 / 下一個」按鈕。  
  * 或使用鍵盤 ← (左) 與 → (右) 鍵切換。

### **C. 搜尋與分類 (Search & Filter)**

* **關鍵字搜尋**：在搜尋框輸入文字，系統會即時篩選符合「中文名稱」或「英文名稱」的卡片。  
* **類別篩選**：點擊上方的黑色/白色膠囊按鈕（如 Environmental），僅顯示該類別的詞彙。點擊 All 則顯示全部。

## **4\. 測驗挑戰 (Quiz Challenge)**

點擊右上角的「**測驗挑戰**」按鈕，系統提供兩種測驗模式來檢視學習成果：

### **模式一：英譯挑戰 (Translation Challenge)**

* **題型**：題目顯示 **中文詞彙**。  
* **目標**：從選項中選出正確的 **英文術語**。  
* **適用對象**：想加強中英名詞對照記憶的使用者。

### **模式二：詞意測驗 (Meaning Challenge)**

* **題型**：題目顯示該詞彙的 **詳細定義/描述**。  
* **目標**：從選項中選出對應的 **英文專有名詞**。  
* **適用對象**：想深度理解詞彙背後含義的使用者。

### **測驗流程：**

1. 每回合隨機抽選 **5 題**。  
2. 答題後系統會立即顯示對錯，若答錯會顯示正確答案與解析。  
3. 測驗結束後顯示總分與評語（如：🏆 ESG 專家就是你！）。  
4. 可選擇「再試一次」或「選擇其他模式」。

## **5\. 常見問答 (FAQ)**

Q: 為什麼聽不到發音？  
A: 發音功能依賴瀏覽器內建的語音合成引擎 (Web Speech API)。

* 請確認您的電腦/手機音量已開啟。  
* 若使用手機，請確認未處於靜音模式。  
* 部分舊版瀏覽器可能不支援此功能，建議使用 Chrome, Edge 或 Safari。

Q: 如何在手機上操作？  
A: 本網頁採響應式設計 (Responsive Design)，在手機上操作皆與電腦相同。分類按鈕區塊可用手指左右滑動來查看更多選項。  
Q: 測驗題目是固定的嗎？  
A: 不是。每次點擊「測驗挑戰」或「再試一次」，系統都會從資料庫中隨機抽選 5 題，確保每次練習都有新鮮感。

## **6\. 鍵盤快捷鍵 (Keyboard Shortcuts)**

在 **學習模式 (Flashcard Mode)** 下，支援以下快捷鍵：

| 按鍵 | 功能 |
| :---- | :---- |
| **空白鍵 (Space)** | 翻轉卡片 (正面/背面) |
| **右方向鍵 (→)** | 下一張卡片 |
| **左方向鍵 (←)** | 上一張卡片 |
| **Enter 鍵** | 翻轉卡片 |
